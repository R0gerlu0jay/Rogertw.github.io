在Terminal上寫程式?!(Neovim)

Roger Lv1
  2025-02-02 16:31:27 2025-02-02 16:31:27 Created   2025-02-04 13:51:17 2025-02-04 13:51:17 Updated  
  • Neovim 
 

為什麼會想研究這個

單純就是因為看起來很帥，看到別人在Terminal上寫程式，感覺很厲害，所以就想要來研究一下
想看安裝的教學影片直接到从零开始Neovim 鼠标我已经扔了

安裝方法

安裝Neovim

可以用系統自帶的apt-get安裝(版本較舊)，也可以用Homebrew安裝(版本較新)，我自己是用Homebrew安裝的，如果沒有安裝Homebrew的話可以參考這裡 ，安裝完Homebrew後就可以用下面的指令安裝Neovim

$ brew install neovim

安裝完後可以用$ nvim指令來啟動Neovim

大概會長這樣，醜醜的，不過可以自己設定主題，字體等等……

安裝插件管理器

Neovim的插件管理器有很多種，想要研究的可以到這個repo 看看，我自己是用LazyVim ，只需要到官網上複製安裝的指令，以下是Linux/Mac的指令

$ git clone https://github.com/LazyVim/starter ~/.config/nvim

輸入完後就可以用$ nvim指令來啟動Neovim，會看到一些提示，等待安裝完就可以開始使用了

安裝好後大概會長這樣


淺色模式和深色模式會變色，所以不要覺得奇怪

如果介面icon的地方都是亂碼的話，那可能是因為系統的默認字體沒有支持這些icon，Linux用戶可以參考這個影片从零开始Neovim 鼠标我已经扔了

使用方法

基本操作

我先建立了一個test資料夾，裡面有一個hello.py檔案，我會先$ cd test進入資料夾，然後用$ nvim .來開啟資料夾（如下圖）

左邊叫做Neo-Tree ，可以看到資料夾的內容，右邊是編輯區，可以用來編輯所選的檔案

快捷鍵

Neovim的快捷鍵和一般的編輯器不太一樣，可以參考這裡 來了解一下，因為秉持可以用滑鼠解決就用滑鼠解決，所以我也沒有特別去記快捷鍵，只有一些基本的，例如

  • i：進入編輯模式
  • esc：退出編輯模式
  • :w：儲存
  • ctrl + /：叫出終端

今天如果碰巧沒有滑鼠，以下是取代滑鼠的快捷鍵

  • h/j/k/l：上/下/左/右
  • ctrl + h/j/k/l：上/下/左/右(移動光標到另一個窗口)

如果今天要編輯隱藏文件(例如.gitignore)，先將光標移動到Neo-Tree上，然後按shift + i，就可以看到隱藏文件了

在編輯窗口上方如果有很多文件，可以用shift + h/l來左右切換文件編輯(可被滑鼠代替)

當編輯完文件並且儲存後需要關閉文件的話可以 “ 依序 “ 按下空白鍵 + b + d

要離開Neovim的話輸入:qa，如果有未儲存的文件的話，可以輸入:qa!強制離開

LSP（Language Server Protocol）

使開發者可以更輕鬆地使用語言伺服器來獲取程式碼補全、語法檢查、跳轉定義、重構等功能。設定方法如下：
在Neovim裡面輸入:LazyExtras打開擴展介面，裡面有很多支持的插件，可以使用x來開啟/關閉擴展插件

在推薦欄裡會檢查你當下的文件所需要的語言插件，我現在要安裝python的語言插件，就用滑鼠點一下或是使用j/k來選擇，光標到python那行後按x來安裝，接著重啟Neovim就可以使用了

如果沒看到自己想要的語言插件，可以輸入/來搜尋，例如我想要安裝C語言的插件，輸入/lang.clangd，然後按下enter之後按x就可以安裝了

記得都要重啟過後語言插件才會安裝並生效


現在就會跳提示詞了！！！

如果說Python的語言插件安裝好後依舊沒有跳提示詞解決方法如下

  • 安裝pyright(lazyvim預設的LSP)，在NeoVim裡面輸入:MasonInstall pyright
  • 安裝fzf(查找文件的工具)，在終端輸入$ brew install fzf

接著再回去編輯python文件，應該就會看到提示詞了

結論

沒什麼結論，只是想要研究一下而已，圖形化介面還是比較好看，Vscode用久了還是比較習慣，但如果沒有滑鼠的話，Neovim還是比較方便，而且也比較不會分心，Vscode的狗狗插件最好別裝(極易分心)，總之就是看個人喜好吧！
